# chinese-chess

#### 介绍
简易中国象棋在线对战

#### 软件架构
B/S架构

使用 SpringBoot + Websocket+ Html + CSS + Javascript 开发


#### 安装教程
直接下载jar包，执行`java -jar chinese-chess-0.0.1-SNAPSHOT.jar`启动项目
项目运行在8086端口，访问地址为：http://localhost:8086

#### 使用说明
拉取代码，可以自行修改代码和重新打包
